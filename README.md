# DSA Sample Programs

Three beginner-friendly C programs demonstrating core data structures.

---

## Programs

### 1. Notification Queue (`queue-notification-system/`)
Demonstrates a **queue** (FIFO) using a fixed-size array.
- Add notifications to the back of the queue
- Process (send) notifications from the front

### 2. Task Manager (`stack-task-manager/`)
Demonstrates a **stack** (LIFO) using a fixed-size array.
- Push tasks onto the top of the stack
- Pop the most recently added task
- View all tasks in stack order

### 3. Music Playlist (`linkedlist-music-playlist/`)
Demonstrates a **singly linked list** using dynamic memory allocation.
- Add songs (name, artist, genre) to the front of the playlist
- Remove a song by name
- View all songs in the playlist

---

## How to Run

1. Open the online C compiler: https://www.programiz.com/c-programming/online-compiler/
2. Copy the contents of `sample.c` from the program you want to run
3. Paste it into the editor, fill the missing code and click **Run**
4. Follow the on-screen menu to interact with the program
5. Implemented code is found in `final.c` files.
6. Algo visualizer project https://algovisualizerproject.netlify.app/
